# Verification

- `pnpm typecheck`
- `pnpm lint`
- `pnpm test`
- `pnpm build`
- `pnpm test:e2e`
- `pnpm licenses`

Structural parity covers ordered flow, authored page breaks, decorations, blank pages, warnings, and page metadata. Browser E2E covers resolver boundaries, limits, lifecycle, security, canonical iframe Viewer behavior, and all installed engines.
