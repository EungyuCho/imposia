## Agent skills

### Issue tracker

Issues are tracked as local Markdown files under `.scratch/<feature>/`. See `docs/agents/issue-tracker.md`.

### Triage labels

Triage uses the canonical `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, and `wontfix` roles. See `docs/agents/triage-labels.md`.

### Domain docs

Domain documentation uses a single-context layout rooted at `CONTEXT.md`, with `docs/routing.md` routing to domain docs and ADRs. See `docs/agents/domain.md`.
