---

이 페이지는 npm에 공개된 버전만 다룹니다. `@imposia/core`, `@imposia/viewer`, `@imposia/client`, `@imposia/react`는 같은 버전 번호를 사용합니다.

## 0.4.0 — 2026년 7월 24일

긴 HTML·CSR 업데이트도 확정되는 페이지 구조를 바꾸지 않으면서 메인 스레드에 실행 시간을 양보합니다.

- `PageDocumentOptions.compose`에 기본 8ms 실행 예산, 주입 가능한 스케줄러, `Infinity` 비활성화 옵션을 추가했습니다.
- 완료 페이지 수와 수렴 패스를 포함하는 잠정 진행률을 추가했습니다.
- 스테이징 콘텐츠와 스타일을 마운트한 뒤 글꼴과 이미지가 준비될 때까지 기다립니다.
- 스케줄러 대기를 중단할 수 있게 하고, 이전 세대를 정리한 뒤 대체 세대를 시작하도록 직렬화했습니다.
- 네이티브 인쇄 스냅샷에서 원본 `html`·`body` 문맥, body 범위 선택자, 계산된 사용자 정의 속성을 보존합니다.
- 페이지를 넘는 카운터가 유지되도록 style containment 없이 확정 페이지 루트에 layout containment를 적용했습니다.

[0.4.0 릴리스 노트 보기](https://github.com/EungyuCho/imposia/releases/tag/v0.4.0)

## 0.3.0 — 2026년 7월 24일

인쇄를 iframe 크기에서 분리하고, 제약된 표 조합을 위한 명시적인 확장 지점을 추가했습니다.

- Chromium 백지 인쇄를 피하도록 네이티브 인쇄를 최상위 문서의 일시적이고 격리된 스냅샷으로 옮겼습니다.
- Core와 Publication 확장을 위한 동기식 `finalizePage` 훅을 추가했습니다.
- 분할된 표의 열 너비를 고정하는 선택형 `createTableColgroupExtension()` 프리셋을 추가했습니다.
- 분할 표, 안전한 Grid, 지나치게 높은 일반 블록이 현재 페이지의 남은 공간부터 사용합니다.

[0.3.0 릴리스 노트 보기](https://github.com/EungyuCho/imposia/releases/tag/v0.3.0)

## 0.2.0 — 2026년 7월 24일

호스트 애플리케이션이 Viewer 조작 UI를 완전히 조합할 수 있게 했습니다.

- 호스트가 탐색·보기 모드·확대 UI를 직접 제공할 수 있도록 <code className="whitespace-nowrap">controls: false</code>를 추가했습니다.
- 즉시 Viewer 상태 구독, React 상태 콜백, 명령형 탐색과 확대 제어를 추가했습니다.
- Viewer 토큰과 표시 스타일을 각 Viewer 루트로 한정해 스타일시트를 가져와도 호스트 페이지가 바뀌지 않습니다.

[0.2.0 릴리스 노트 보기](https://github.com/EungyuCho/imposia/releases/tag/v0.2.0)

## 0.1.3 — 2026년 7월 23일

HTML·CSR 페이지네이션 정합성을 제품의 주 계약으로 정했습니다.

- 원본 토큰이 확정 페이지 전체에서 정확히 한 번씩 순서대로 나타나는지 브라우저별로 검증합니다.
- canonical iframe을 유지하면서 빠른 소스 변경을 적용하는 공개 CSR 정합성 예제를 추가했습니다.
- reflowable EPUB을 주 렌더링 경로가 아닌 지원되는 시맨틱 투영으로 명확히 했습니다.

[0.1.3 릴리스 노트 보기](https://github.com/EungyuCho/imposia/releases/tag/v0.1.3)

## 0.1.2 — 2026년 7월 23일

릴리스 검사와 데모 서버의 보안을 강화했습니다.

- 검증과 코드 스캔에 CodeRabbit 자동 리뷰를 추가했습니다.
- 사용하지 않는 데모 서버 요청 지연 기능과 관련 CodeQL 경고를 제거했습니다.
- 구조 선택자는 보수적인 대체 동작을 갖는 `Element.matches()`에만 전달됨을 확인했습니다.

[0.1.2 릴리스 노트 보기](https://github.com/EungyuCho/imposia/releases/tag/v0.1.2)

## 0.1.1 — 2026년 7월 23일

데모, 문서, 릴리스 경로를 공개 패키지 업데이트에 맞게 정비했습니다.

- 세로 A4 기본값과 방향 전환, A3·A4·B1 용지 프리셋을 추가했습니다.
- 다국어 API와 퍼블리싱 문서를 확장했습니다.
- npm trusted publishing, 태그, GitHub Release, tarball, 체크섬을 검증하는 보호된 릴리스 경로를 추가했습니다.

[0.1.1 릴리스 노트 보기](https://github.com/EungyuCho/imposia/releases/tag/v0.1.1)

## 0.1.0 — 2026년 7월 19일

최초 브라우저 전용 릴리스에서 공개 패키지 구성을 완성했습니다.

- 하나의 영속적인 canonical iframe에서 결정적인 브라우저 페이지네이션을 제공합니다.
- Viewer 표시 계층, 브라우저 Client 진입점, 기본 React 어댑터를 추가했습니다.
- Publication 조합, Reader 탐색·검색·썸네일과 여러 보기 모드를 추가했습니다.
- 확정 소스에서 네이티브 인쇄와 reflowable EPUB 3.3 내보내기를 제공합니다.
- 타입 진단, 제한된 확장, 자산 해석과 표·Flex·Grid·다단·CJK 분할의 지원 범위를 정했습니다.

[npm에서 @imposia/core 0.1.0 보기](https://www.npmjs.com/package/@imposia/core/v/0.1.0)
